# Jobpac Mobile — Product Overview

A walkthrough website for Jobpac Mobile, an extension of the Jobpac ERP that enables accurate, real-time time tracking for field workers and crew leads.

## Adding Prototype Screenshots

Place your screenshots in the `public/prototypes/` folder using these filenames:

| File | Screen |
|------|--------|
| `dashboard.png` | Landing Screen — Dashboard |
| `job-time-entry.png` | Job Time Entry |
| `crew-timesheet.png` | Crew Timesheet — Default Hours |
| `pte.png` | Pre-Tax Entry (PTE) |

## Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

This site is deployed via [Vercel](https://vercel.com). Any push to the `main` branch automatically triggers a new deployment.

To set up Vercel deployment:
1. Go to [vercel.com](https://vercel.com) and sign in
2. Click **Add New Project**
3. Import the `nawaz-md-trmb/myrepo` repository from GitHub
4. Vercel auto-detects Next.js — click **Deploy**

All future pushes to `main` will deploy automatically.
